# 待办事项应用 (Todo App)

一个使用原生 HTML、CSS、JavaScript 开发的简单待办事项应用。

## 项目结构

```
XU-hw-2/
├── index.html      # HTML 页面结构
├── style.css       # CSS 样式文件
├── script.js       # JavaScript 功能逻辑
└── README.md       # 项目说明文档
```

## 功能特性

### 基本功能
- ✅ **添加待办事项** - 在输入框中输入任务内容，点击"添加"按钮或按回车键添加
- ✅ **标记完成** - 点击"完成"按钮将任务标记为已完成（文字划掉效果）
- ✅ **取消完成** - 再次点击"取消"按钮可以取消完成状态
- ✅ **删除任务** - 点击"删除"按钮从列表中移除任务

### 筛选功能
- ✅ **全部** - 显示所有待办事项
- ✅ **未完成** - 只显示未完成的待办事项
- ✅ **已完成** - 只显示已完成的待办事项

### 批量操作
- ✅ **清除已完成** - 一键清除所有已完成的待办事项
- ✅ **清除全部** - 清除所有待办事项（有确认提示）

### 数据持久化
- ✅ **LocalStorage 存储** - 所有数据自动保存到浏览器本地存储
- ✅ **自动加载** - 页面刷新后自动恢复之前的待办事项

## 技术实现

### HTML (index.html)
- 使用语义化标签（`<header>`, `<main>`, `<section>`, `<footer>`）
- 结构清晰，易于维护
- 无障碍访问友好

### CSS (style.css)
- 使用 CSS 变量实现主题配色
- 采用 Flexbox 布局
- 响应式设计，支持移动端
- 丰富的悬停和过渡动画效果
- 最大宽度 800px，居中显示

### JavaScript (script.js)
- 原生 JavaScript，无依赖
- 模块化函数设计
- 完整的中文注释
- LocalStorage 数据持久化

## 使用方法

1. 直接用浏览器打开 `index.html` 文件即可使用
2. 或者使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8080
   
   # 或使用 Node.js
   npx serve
   ```

## 快捷键

- `Enter` - 在输入框中按回车键添加待办事项
- `Ctrl/Cmd + Enter` - 快速添加待办事项
