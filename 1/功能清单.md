## 1. 功能清单

### 1.1 系统概述

TaskFlow 是一款科技风格的项目任务管理系统前端原型，采用 React 框架开发，支持 PC 端和移动端自适应布局。系统包含四大核心模块：控制台、项目管理、任务中心、预警中心。

### 1.3 数据模型

#### 项目数据结构
```javascript
{
  id: Number,           // 项目ID
  name: String,         // 项目名称
  department: String,   // 所属部门
  status: String,       // 状态: progress|completed|delayed|pending
  leader: String,       // 负责人
  members: Array,       // 成员列表
  startDate: String,    // 开始日期
  endDate: String,      // 结束日期
  progress: Number,     // 进度百分比 0-100
  milestones: Array,    // 里程碑列表
  currentMilestone: Number  // 当前里程碑索引
}
```

#### 任务数据结构
```javascript
{
  id: Number,           // 任务ID
  name: String,         // 任务名称
  project: String,      // 所属项目
  assignee: String,     // 负责人
  status: String,       // 状态: progress|completed|delayed|pending
  priority: String,     // 优先级: high|medium|low
  progress: Number,     // 进度百分比 0-100
  startDate: String,    // 开始日期
  endDate: String,      // 结束日期
  description: String   // 任务描述
}
```

#### 预警数据结构
```javascript
{
  id: Number,           // 预警ID
  type: String,         // 类型: deadline|overdue|progress|milestone|resource
  level: String,        // 级别: critical|high|medium|low
  title: String,        // 标题
  message: String,      // 消息内容
  project: String,      // 关联项目
  assignee: String,     // 相关负责人
  time: String,         // 时间
  read: Boolean         // 是否已读
}
```

### 3.5 部署到服务器

#### 方式一：静态文件部署

```bash
# 1. 构建项目
npm run build

# 2. 将 dist/ 目录内容上传到 Web 服务器
# 支持: Nginx, Apache, Caddy, 阿里云OSS, 腾讯云COS 等
```

**Nginx 配置示例**:
```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /var/www/project-manager/dist;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # 启用 gzip 压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript;
}
```

#### 方式二：Docker 部署

**Dockerfile**:
```dockerfile
# 构建阶段
FROM node:20-alpine as build
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

# 生产阶段
FROM nginx:alpine
COPY --from=build /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

```bash
# 构建镜像
docker build -t taskflow:latest .

# 运行容器
docker run -d -p 80:80 taskflow:latest
```

#### 方式三：Vercel/Netlify 一键部署

1. 将代码推送到 GitHub/GitLab
2. 登录 Vercel 或 Netlify
3. 导入项目仓库
4. 配置构建命令: `npm run build`
5. 配置输出目录: `dist`
6. 点击部署
